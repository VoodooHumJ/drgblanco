Dr. Guido Blanco - Sitio Web Profesional
=========================================

DESCRIPCIÓN:
Sitio web profesional del Dr. Guido Blanco, especialista en cirugía de columna y escoliosis. 
Incluye información sobre servicios médicos, experiencia profesional y datos de contacto.

CARACTERÍSTICAS PRINCIPALES:
- Sección de servicios y procedimientos con iconos interactivos
- Integración de WhatsApp para contacto directo
- Mapa de ubicación en Google Maps
- Diseño responsivo adaptado a dispositivos móviles
- Animaciones CSS y efectos de desplazamiento

TECNOLOGÍAS UTILIZADAS:
- HTML5 semántico
- CSS3 (Flexbox, Grid)
- JavaScript vanilla
- Font Awesome 5 (iconos)
- Plugin AOS (Animate On Scroll)

INSTALACIÓN:
1. Clonar repositorio:
   git clone https://github.com/tu-usuario/dr-guido-blanco.git

2. Instalar dependencias:
   npm install

3. Ejecutar servidor local:
   npm start

PERSONALIZACIÓN:
- Para cambiar colores: modificar variables CSS en `css/main.css`
- Actualizar información de contacto en `index.html` (líneas 650-654)
- Modificar descripciones de servicios en `index.html` (sección #services)
- Reemplazar imágenes en `images/` manteniendo nombres y formatos
